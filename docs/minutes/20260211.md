# 2차 프로젝트 2팀 회의록

- 참여자: 윤상이 (팀장), 현길용, 김희성, 정창규, 정상원
- 버그는 우리의 친구 버그는 우리의 동반자 => 강사님의 명언.

## 오전 회의 내용

- 맡은 작업 마무리 및 검토 작업 확실시 할 것

## 작업 목표

- 게시판 컨트롤러 / 시큐리티 마무리 작업 및 검토
- 게시물 코드 작업은 마무리 검토 작업 마무리
- 내일 메인페이지 / 게시물 부분 작업 목표 잡음
- 로그인 유저 부분 마무리 

## 작업 과정

- User 부분 마무리 (Signup, Login, Passkey, Email), message Entity 제작
- Post 부분 마무리 (Service, Controller), board & User Entity에 맞춰서 수정 작업 실시
- Board 부분 마무리 (Controller, Config), User data 받아서 수정 작업 실시
- header 부분과 Side bar 부분 마무리, Status 저장, Signup Login Page 작성중

## 작업 회고

- 이메일 발송 로직간의 보안문제 때문에 접근이 안되서 발송이 안되었는데, 
  페이지 접근방식을 우선순위를 두어 해결했습니다.
- 지식이 없다보니 도구에 의지하는 모습이 초라해 보였다.
- 보드 부분 마무리까지는 도구에 의지하여 완성하였으나 
  유저와 연결하여 하려다보니 어떤 순서로 진행해야하는지 막막했다.
- 처음 작성시 예상한 분량 중 프론트 부분 작성을 하다보니 중간중간 늘어나는 
  분량 때문에 작업량 조절이 어렵다.
