# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 2월 12일
- 참여자: 윤상이 (팀장), 현길용, 김희성, 정창규, 정상원
- 버그는 우리의 친구 버그는 우리의 동반자 => 강사님의 명언.

## 오전 회의 내용

- 각 자 맡은 부분 마무리 후에 오후에 Javascript를 통해서 
  프론트 엔드와 백 엔드를 연결하려고 했다.

## 작업 목표

- 팝업(쪽지)부분 완성 단계 예정.
- 게시물과 유저 연결 마무리 및 테스트
- 보드와 유저 연결 마무리
- 회원(Form)부분 마무리

## 작업 과정

- 프론트 엔드와 백 엔드 사이 문제점 발생하여 연결하지 못하여 수정중.
- Post 작업 마무리 후 테스트 진행
- 작업하고 있는 코드들을 재확인 후 수정하고 있다.
- Board와 유저 연결 작업진행 

## 작업 회고

- 엔티티 작성 중 서로 다른 엔티티 간의 충돌을 예상하지 못해 문제가 발생했습니다. 
  이를 해결하기 위해 중첩되는 내용을 별도의 엔티티로 분리하여 수정했습니다.
- 게시물 작업하면서 권한 관리에 대해 공부할 필요가 있다 생각했습니다. 
- Board와 User 게시판 연결하는 것에 대해 어려움을 느꼈다.
- Back End가 생각보다 복잡하다는 것을 알게되었다.

## 내일 목표

- 쪽지 부분 완성
- 댓글 작업 시작 
- 게시판 마무리 및 첨부파일 작업 시작
- Back End와 연결시켜보는 것이 목표
