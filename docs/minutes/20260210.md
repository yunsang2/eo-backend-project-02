# 2차 프로젝트 2팀 회의록

- 날짜: 2026년 2월 10일
- 참여자: 윤상이 (팀장), 현길용, 김희성, 정창규, 정상원
- 버그는 우리의 친구 버그는 우리의 동반자 = 강사님의 명언.

## 오전 회의 내용

- 사이드 바 들어갈 내용 정할 것
- 데이터베이스 구축 후 코드 작성 들어갈 예정 

## 작업 목표

- 공통 css 작업 마무리와 로그인 페이지 프론트 작업 시작
- 데이터 베이스 작업 마무리
- 시간이 된다면 객체 생성 할 예정

## 작업 과정

- 사이드 바 내용 상의 후 프론트 엔트 작업자에게 전달 
  - 로그인 / 회원가입 마무리 
  - 메인페이지 어느정도 작성 
  - 게시판 틀 구축
- 데이터베이스 요소 작성 마무리 
  - 회의한 내용에 있는 요소는 넣어놓을 예정
  - 게시판(정창규), 게시물(현길용), 사용자(윤상이)  작성

## 작업 회고

- 프론트 엔드 진행 중 Thymeleaf layout dialect란 것을 새롭게 알게 되었다.
- 게시물 작업 중 정답인지 아닌지 어떻게 검증해야할지 모르겠고 어려움을 많이 느꼈다.
- 게시판 작업 중 많은 부분이 겹치며 연결부분에서 매끄럽지 않아 많이 당황하였고 어려움을 많이 느꼈다.
- 작성 순서를 지키지 않고 작성해서 생각지도 못한 오류가 나오니 대처할 수 없었다. 이후 강사님에게 질문을 
  통해 해결했지만 스스로의 힘으로 해결할 수도 있었던 상황인데도 불구하고 해결하지 못해 앞으론 복습과 
  작성 순서를 지키며 작업해야겠다고 생각했다.

## 내일 목표

- 게시판 컨트롤러 / 시큐리티 마무리 작업 및 검토
- 게시물 코드 작업은 마무리 검토 작업 마무리
- 내일 메인페이지 / 게시물 부분 작업 목표 잡음
- 로그인 유저 부분 마무리
